# UrbanElectra 🛒⚡

A full-stack e-commerce platform for electronics built with the MERN stack.

## 🚀 Live Demo
[https://urban-electra.vercel.app/]

## ✅ Project Status
**Completed** — Core features are fully implemented and deployed.
Maintenance will be performed as needed.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Zustand
- **Backend:** Node.js, Express, MongoDB Atlas
- **Payments:** Stripe
- **Storage:** Cloudinary, Upstash Redis
- **Deployment:** Vercel

## ✨ Features
- User authentication with JWT & refresh tokens
- Product catalog with categories
- Shopping cart
- Stripe checkout
- Coupon/discount system
- Admin dashboard with analytics
